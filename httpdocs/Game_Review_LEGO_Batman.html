This is one of my favorite Travellers Tales games.
LEGO Batman is based on Batman: The Animated series,
and has it's own unique story.
From a Builder's perspective, my favorite feature is the character creator,
which allows you to create custom heroes and villains.
Some of the pieces in the character creator can be used to create
some of the characters from the comics, such as The Spoiler,
which might be interesting to DC fans.
Another thing DC fans might be interested in are the
unlock-able "facts" in the game's hub, which feature details
about the character of the Batman comics.

LEGO Batman was the first Travellers Tales PC LEGO game to exceed
the limit of 18 story levels (LEGO Batman has 30.)
One feature that sets LEGO batman apart from other games is the
abundance of story levels played from the antagonist's viewpoint.
However, LEGO Batman doesn't seem to have as many characters as
other Travellers Tales games.

However, the main issue I have with the PC version of this game is a glitch
which prevents me from unlocking a certain character. I do not think this
glitch is present in the console versions of this game.
